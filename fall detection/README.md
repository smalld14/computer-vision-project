
# Fall detection

模型說明
--
-使用YOLOv8 pose estimation model 取得人體的17個關節點(keypoint)
-觀察人體跌倒時關節點特徵，並根據特徵開發跌倒演算法
-主要使用頭部、四肢、臀部關節點的相對位置作為判斷根據
